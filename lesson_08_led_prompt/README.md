# Lesson 8 — LED via prompts (Groq)

**Edit only:** `my_program.py`

Send test phrases to Groq; model replies **LED_ON**, **LED_OFF**, or **NONE**.

## Groq key

Create a key at [console.groq.com](https://console.groq.com) → `.env`:

```
GROQ_API_KEY=gsk_...
```

## Run

```bash
cd ~/lessons
./run 8
./reset 8
```
