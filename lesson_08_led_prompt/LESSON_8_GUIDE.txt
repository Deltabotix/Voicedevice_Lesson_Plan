Lesson 8 — LED via prompt engineering
=====================================

Step 1: Understand it
---------------------

How does it work?

You write **test sentences** in Python. Each goes through a **prompt** to **Groq** (AI).
The model must answer with **one word**: **LED_ON**, **LED_OFF**, or **NONE**.
Your code drives the **real red LED** — so you **see** when the prompt worked.

This is **prompt engineering**: the words you give the model change what it does.

What will you need to do?

1. Add **GROQ_API_KEY** to `.env` (your own key from console.groq.com).
2. Edit **PROMPT_TEMPLATE** and **TEST_PHRASES** in `my_program.py`.
3. Run `./run 8`.

Real-world use

Voice kits route “turn on the light” vs “what’s the weather?” through classifiers like this.


Step 2: Create it
-----------------

Step A — Groq key

  cp ~/lessons/.env.example ~/lessons/.env
  nano ~/lessons/.env
  # GROQ_API_KEY=gsk_...

Step B — settings in my_program.py

  LED_PIN = 26
  LED_ON_LEVEL = "HIGH"
  PROMPT_TEMPLATE = ...   # must include {utterance}
  TEST_PHRASES = [
      "turn on the led",
      "switch the light off",
      "what is two plus two",
  ]

Step C — my_program()

  helper.check_prompt_lab_ready(TEST_PHRASES, PROMPT_TEMPLATE)
  helper.pick_how_led_turns_on(LED_ON_LEVEL)
  helper.prepare_led(LED_PIN)
  helper.light_off()

  for phrase in TEST_PHRASES:
      label = helper.classify_led_request(phrase, PROMPT_TEMPLATE)
      print(f"{phrase!r} -> {label}")
      helper.apply_led_label(label)
      helper.pause(0.4)

Step D — run

  cd ~/lessons
  ./run 8


Step 3: Test and improve it
---------------------------

Success

  • ON phrases → LED on; OFF phrases → off; chat → NONE (no change).
  • You can explain one wrong answer and how to fix the prompt.

Troubleshooting

  • Missing key → check .env and GROQ_API_KEY.
  • Model adds extra words → tighten PROMPT_TEMPLATE (“exactly one word”).

Go further

  • Add polite vs blunt phrasing to TEST_PHRASES.
  • Add trick questions that should be NONE.

Reset

  cd ~/lessons
  ./reset.sh 8
